## Scope

This repository records the formal expansion of the AI Visibility canonical definition into a structured set of dependent theorems. Each theorem assumes and inherits from the AI Visibility Canonical Definition and does not modify or replace it.

AI Visibility is a systems discipline concerned with how information is authored, structured, and emitted so it can be reliably ingested, retained, and recalled by large language models with minimal semantic ambiguity across training and inference cycles.

All theorems in this repository operate upstream of prompting, ranking, retrieval, and any post-training optimization.
